工作进展

本周我们着重调研了当前存在的sfm算法，并对他们做了一个分析与总结。
当前主流的sfm算法有sequtial sfm，hierarchical sfm以及global sfm。在Richard Hartley写的《Multiview Geometry in Computer Vision》里面着重强调的是sequential sfm算法。就是我们所说的主流的sfm算法。

大致的流程图如下：

然而由于误差的累积，会出现drift error的问题。如下图所示：

因此，便需要如同SLAM技术所需要的闭环检测。
然而随着大数据时代来临，sequential sfm这种添加一帧估计一帧相机参数的方式效率低下，尽管在特征匹配以及bundle adjustment里应用了gpu加速技术，然而效率仍旧低下。因而近些年又产生了hierarchical sfm以及global sfm。

由此我们可见在初始场景确定了之后，hierarchical sfm进行了分层添加图片方法，先对子场景做一个融合，然后再进行姿态估计和场景重构。而global sfm则对所有图片一视同仁，直接将所有图片输入，算出他们两两之间的两视图几何几何关系，然后先解决旋转再解决平移，最后对初始结构和场景做一个整体的光束平差。这样做的好处避免在sfm选取初始结构时花费大量的精力，同时对误差能够做一个平均分摊，因而有效地避免了drift error的问题。

下步计划

由上面分析可知，global sfm不仅在精度上有所提升，在效率上由于只做一次bundle adjustment并且采用图模型来求解外参，因而极大的提升了效率。所以我们最终决定采用global sfm作为最终的算法。下周我们将实现算法并争取在数据库上跑通算法。

Quick Start

3D reconstruction from multiple images is the creation of three-dimensional models from a set of images. It is the reverse process of obtaining 2D images from 3D scenes.
The essence of an image is a projection from a 3D scene onto a 2D plane, during which process the depth is lost. The 3D point corresponding to a specific image point is constrained to be on the line of sight. From a single image, it is impossible to determine which point on this line corresponds to the image point. If two images are available, then the position of a 3D point can be found as the intersection of the two projection rays. This process is referred to as triangulation. The key for this process is the relations between multiple views which convey the information that corresponding sets of points must contain some structure and that this structure is related to the poses and the calibration of the camera.
